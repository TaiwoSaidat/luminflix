## Lumiflix Movie App

This is a React(Next) project displaying movies froma a backend source the transactions dashboard with filters enabled for transaction status. Includes a authentication page, hhom page displaying movies and filtered movies absed on topics.

### Problem solved

- Displays movies, categorises them and filters through be relevant topics

### Technologies used

### Frontend Framework & Libraries

- Next.js 16.1.1 - React framework with App Router
- React 19.2.3 - UI component library
- TypeScript 5 - Type-safe JavaScript
- Tailwind CSS 4 - Utility-first CSS framework

### Development Tools

- ESLint - Code linting and quality
- Node.js - JavaScript runtime

### Design Decisions

Component Architecture - Modular, reusable components
Type Safety - Full TypeScript implementation
CSS-in-JS - Tailwind utility classes for styling
Client-Side Rendering - React hooks for state management
